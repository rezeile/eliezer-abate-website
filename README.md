# eliezerabate.com

Personal website for Eliezer Abate — engineer, writer, and builder.

## About

This site showcases my work on Ethiopia, including interactive maps and visual essays published on [GraspWell](https://graspwell.com), along with an archive of writings from 2020–2022.

## Tech Stack

- **Framework**: Next.js with TypeScript
- **Styling**: Tailwind CSS
- **Content**: MDX for blog posts
- **Deployment**: Vercel

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Structure

- `/src/pages` — Next.js pages (home, blog archive, individual posts)
- `/content/blog` — MDX blog posts
- `/public/images` — Static images for blog posts
